# dumbfi

An incomplete collection of dumb financial models and tools.

## Tools

- **[rye](https://github.com/astral-sh/rye)**: a Python package manager

## Build and Run

To build and run the project, follow these steps:

1. **Build the project:**

```bash
cargo build
```

2. **Run the project:**

```bash
cargo run -- --config demo.toml
```

3.  **Run the tests:**

```bash
cargo test
```
